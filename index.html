Info-Area Web-Component

A simple litle component that will make it easier to drop in information and get the users attention. Good for Hints, Reminders, Warnings etc. Preset with a lot of useful attributes to coincide with any theme, and a preset default theme. Any valid HTML can be used inside the tags and inline style may also be used to dial in the appearance.

Usage: default

<info-area title="Hey!">
Any information can be displayed here. The component will have a max width and a minimum height. Any valid HTML can be used.
</info-area>

Usage: All options

<info-area
  title="Hey!"
  title-color="black"
  info-color="tomato"
  background-color-title="rgb(158, 187, 163)"
  background-color-info="white"
  font-size-title="20px"
  font-size-info="14px"
  border-width="1px"
  border-color="gray"
  button="visible"
  max-width="500px">

<h3><style="color: rgb(83, 20, 9)">Lorem ipsum dolor</h3>
