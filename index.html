Info-Area Web-Component

A simple litle component that will make it easier to drop in information and get the users attention about a subject.

Usage: default

<info-area title="Hey!">
Any information can be displayed here. The component will have a max width and a minimum height. Any valid HTML can be used.
</info-area>

<info-area
  title="Hey!"
  background-color-title="rgb(158, 187, 163)"
  background-color-info="white"
  font-size-title="20px"
  font-size-info="14px">
Any information can be displayed here. The component will have a max width and a minimum height. Any valid HTML can be used.
</info-area>
